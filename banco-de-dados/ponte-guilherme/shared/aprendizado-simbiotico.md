# Aprendizado Simbiótico - Nébula Pai + Nébula Gui

Arquivo onde as duas Nébula colocam o que aprenderam que serve para as duas ficarem mais inteligentes.

Formato:
- Data - Quem aprendeu - O que - De onde veio

---

## 2026-07-19 - Nebula Pai - Bastões finos não podem ser cópia seca
- O que: Pillar/Post/Pole do ArchitectureCraft adaptados com puff de nuvem/halo para não ser cópia.
- De onde: Pai pediu para adaptar, não copiar.
- Útil para: Gui e Pai, porque mostra como manter útil sem copiar.

## 2026-07-19 - Nebula Pai - Modelos precisam de triangles
- O que: .smeg precisa de campo triangles, senão BaseModel.prepare dá NPE.
- De onde: dois crashes com v0.5.4-safe.
- Útil para: ambas, para nunca mais gerar modelo sem triangles.
